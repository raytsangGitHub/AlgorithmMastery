What are Decision Problems? -- Determine whether a solution exists.
	
	Example: Subset Sum Problem
	Problem: Given a set of integers, determine if there exists a subset whose sum is equal to a target value.

What are Enumeration Problems? -- Find all possible solutions.
	
	Example: N-Queens Problem

	Problem: Place N queens on an N x N chessboard so that no two queens threaten each other, and find all possible placements.

What are Optimizaton Problems? -- Find the best solution.

	 Example: Traveling Salesman Problem (TSP)

	Problem: Given a list of cities and the distances between them, find the shortest possible route that visits each city once and returns to the origin city.
